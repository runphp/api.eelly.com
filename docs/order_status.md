### 返回数据字段说明

字段名              | 类型    | 说明
-------------------|-----------|-------
title              | string   | 订单状态标题
text               |  string        | 订单状态描述
countDown          | int |       倒计时(秒)
countDownTpl       |string |  倒计时模板(模板变量 {time})
express            |string |  快递信息
expressTime        | int | 快递信息的时间戳(秒)
note               | string | 留言
actions            | list | 支持的操作<返回数据actions说明>


> 返回数据actions说明

名称  | 含义
---------------------|----------
cancel               |    取消订单
pay                  | 立即付款
review               | 去评价
notifySendGoods      | 提醒发货
